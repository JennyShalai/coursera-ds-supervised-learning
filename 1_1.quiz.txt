1. Рассмотрим признак "Число обращений клиента в службу поддержки банка". Он принимает только целые неотрицательные значения. Какой тип имеет данный признак?
+ Вещественный — ведь целые числа тоже являются вещественными 
(Логично, что этот признак принимает натуральные значения, а натуральные числа полностью содержатся в вещественных.)
- Категориальный — он ведь может принимать значения лишь из конечного множества.
- Бинарный — почему бы нет.

2. Выберите верные утверждения про признаки.
+ Предсказания для объектов делаются на основе значений признаков.
(Алгоритмы принимают на вход именно признаковые описания объектов.)
+ Признаки описывают объекты в формате, с которым легко работать на компьютере.
(Для этого и нужно признаковое описание объектов — чтобы переводить какое-то абстрактное описание в более формальное.)
- Признаки не используются в задачах кластеризации и поиска аномалий.
(Признаки используются во всех задачах машинного обучения.)
+ Набор значений признаков на объекте представляет собой вектор определённой размерности.
(Полный набор признаков для объекта удобно задавать с помощью вектора, размерность которого совпадает с числом признаков.)
- Признаки задаются только вещественными числами.
(Признаком может быть, например, элемент упорядоченного множества.)

3. Выберите вещественные признаки из списка
+ Температура воздуха
- Город, в котором прописан клиент
(Это категориальный признак.)
- Наличие у клиента банка военного билета
(Признак принимает два значения (да или нет) и является бинарным.)
+ Количество детей
+ Год рождения

4. Выберите признаки, которые могут рассматриваться только как категориальные (и не могут рассматриваться как бинарные или вещественные).
- Скорость интернет-соединения
(Это вещественный признак.)
+ Тарифный план клиента мобильного оператора
(Тарифных планов много, и вряд ли все из них можно сравнивать.)
+ Тип дома — кирпичный, блочный, панельный и т.д.
(Тут конечное число вариантов и уже приведено больше двух.)
- Наличие у клиента банка военного билета
(Это бинарный признак.)
+ Цвет автомобиля
(Цветов бывает больше двух.)
- Год рождения
(Это вещественный признак.)

5. Выберите из списка задачи классификации.
+ Предсказание тарифного плана, который клиент захочет себе подключить.
(Возможных тарифных планов конечное число, поэтому задача относится к задачам многоклассовой классификации.)
- Определение стоимости одежды по фотографии.
(Стоимость является вещественным числом, поэтому задача относится к задачам регрессии.)
- Поиск групп схожих пользователей мобильного оператора.
(Это задача кластеризации, поскольку требуется сгруппировать объекты по некоторой мере сходства.)
- Определение возраста человека по его активности в сети.
Возраст — вещественная переменная, поэтому данная задача относится к задачам регрессии.

6. Выберите верные утверждения про машинное обучение
+ Алгоритм (или модель) — это функция, которая принимает на вход признаковое описание объекта и выдает некоторое предсказание ответа
+ В задачах обучения с учителем требуется построить алгоритм, который по признаковому описанию объекта предсказывает некоторый ответ.
- Функционал ошибки определяет уровень шума в признаках.
(Функционал ошибки определяет качество предсказания алгоритма на выборке.)
- Алгоритм (или модель) — это функция, которая принимает на вход признаковое описание объекта и его ответ, и выдаёт качество предсказаний для данного объекта.
(Алгоритм — это функция, которая по объекту выдает предсказание ответа.)
+ Функционал ошибки позволяет определить, насколько данный алгоритм подходит для решения задачи на конкретной выборке.
